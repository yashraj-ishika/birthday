# 🌸 Happy Birthday Ishika — Romantic Birthday Website

A beautiful, emotional, and interactive birthday website created for Ishika's 20th birthday.

---

## ✅ Completed Features

### 🎨 Design
- Soft pink, pastel, and warm-tone theme throughout
- Elegant typography: **Playfair Display**, **Dancing Script**, **Lato**
- Glowing effects on the heart, title, and letter card
- Responsive and mobile-friendly layout

### 💕 Landing Screen
- Animated, pulsing **heart shape** in the center
- Floating heart particles (💕, 🌸, ✨, 💗) in the background
- Subtle "I love you / Forever / My Ishika ❤️" floating text
- "Click to open" hint with blinking animation

### 💖 Heart Opening Interaction
- Smooth heart-expand animation on click
- Transitions to the birthday section seamlessly
- Auto-plays background romantic music on heart click

### 🎂 Birthday Section
- Main title: **"HAPPIEST 20th BIRTHDAY ISHIKAAA 🫶🏻🫂💋"**
- Birthday message with personal note
- **Heart-shaped photo collage** with 4 photos (SVG clip-path heart shape)
- Each photo is tappable — opens a **beautiful modal** with the romantic caption

### 📸 Photo Captions (Modal)
1. "Kissing you feels like time pauses..."
2. "When I first saw you, everything felt a little softer..."
3. "The first time I held your hand..."
4. "I like admiring you when you're not looking..."

### 💌 Love Letter Section
- Styled as a glowing, animated letter/card
- Full romantic message with elegant typography
- Soft inner glow and animated box-shadow

### 🎁 Surprise Section
- "Click for Surprise 💌" button
- Triggers an **explosion of hearts** animation across the screen
- Reveals: **"You are my everything ❤️"**

### 🎵 Background Music
- Romantic instrumental background music
- **Play/Pause** button (bottom-right corner)
- Gracefully hides if audio fails to load

### ✨ Extra Effects
- **Cursor sparkle trail** on desktop (hearts, flowers)
- **Floating love texts** throughout the page
- **Animated page title** cycling through romantic messages
- Smooth scroll & section fade-in animations

## 🚀 Entry Point

Open `index.html` in any browser. To publish, use the **Publish tab**.
